# 🎯 INICIO RÁPIDO - Sistema de Autoevaluación

## ¿Qué es esto?

Un sistema completo de autoevaluación con **35 preguntas en 7 temas** para los Módulos 2 y 3 del proyecto PFG.

---

## ⚡ 3 Pasos para Empezar

### 1️⃣ **Soy Estudiante**
```
→ Lee: GUIA_USUARIO_AUTOEVALUACION.md
→ Accede a: /modulo2/Tema1/index.html
→ Baja hasta: Sección "AUTOEVALUACIÓN"
→ Responde: Las 5 preguntas
→ Presiona: "Corregir"
→ Ves: Tu calificación
```

### 2️⃣ **Soy Desarrollador**
```
→ Lee: GUIA_RAPIDA_DESARROLLADORES.md
→ Edita: js/questions-data.json (para preguntas)
→ Edita: css/styles.css (para estilos)
→ Prueba: test-autoevaluacion.html
```

### 3️⃣ **Soy Coordinador**
```
→ Lee: AUTOEVALUACION_README.md
→ Verifica: CHECKLIST_IMPLEMENTACION.md
→ Comunica: A estudiantes que está disponible
→ Monitorea: Que todo funciona
```
