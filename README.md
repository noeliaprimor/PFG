## Descripción

Este proyecto desarrolla un recurso docente web orientado a la enseñanza online de infraestructuras computacionales para el procesamiento de datos masivos.

El contenido se centra en tecnologías del ecosistema Big Data, especialmente Apache Spark, junto con arquitecturas y herramientas relacionadas como Hadoop, Kafka, Spark Streaming, Structured Streaming, Docker y entornos de ejecución distribuida.

## Objetivo

El objetivo principal es proporcionar materiales didácticos organizados en módulos, temas y lecciones, combinando:

- contenidos teóricos;
- ejemplos prácticos;
- laboratorios reproducibles;
- ejercicios propuestos;
- recursos y lecturas complementarias;
- autoevaluaciones.

## Memoria del TFG

La memoria del Trabajo Fin de Grado se encuentra en la carpeta Memoria


## Estructura general

El proyecto está organizado en módulos:

- `modulo1`: ecosistema Hadoop y fundamentos de Big Data.
- `modulo2`: procesamiento paralelo en memoria con Apache Spark.
- `modulo3`: gestión de datos en tiempo real, Kafka y Spark Streaming.

Además, incluye recursos comunes para la web:

- `css/`: hojas de estilo.
- `js/`: scripts de navegación, autoevaluación y soporte.
- `partials/`: cabecera, pie de página y componentes reutilizables.
- `img/`: recursos gráficos.

## Uso de Inteligencia Artificial

El proyecto documenta el uso de herramientas de IA como apoyo técnico y editorial.

Los documentos principales son:

- `USO_IA.md`: descripción general del uso de IA en el proyecto.
- `REGISTRO_USO_IA.md`: registro de cambios e interacciones asistidas por IA.
