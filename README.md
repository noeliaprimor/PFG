## Resumen del proyecto

Este repositorio contiene el material desarrollado para un Proyecto de Fin de Grado orientado a la docencia online de infraestructuras computacionales para el procesamiento de datos masivos. El proyecto construye una plataforma web educativa con contenidos teóricos, ejercicios prácticos, notebooks ejecutables y actividades de autoevaluación sobre tecnologías Big Data, con especial atención a Apache Hadoop, Apache Spark, Apache Kafka y el procesamiento de datos en batch y streaming.

El objetivo principal es ofrecer un recurso formativo estructurado, que permita al estudiante avanzar de forma progresiva desde los fundamentos de Big Data hasta la implementación de laboratorios prácticos con Spark y Kafka. Los materiales combinan explicaciones conceptuales, ejemplos guiados, prácticas reproducibles y cuestionarios de evaluación para reforzar el aprendizaje autónomo propio de la enseñanza online.

## Contenidos principales

- **Módulo 2: Procesamiento paralelo en memoria: Apache Spark.** Cubre la arquitectura de Spark, su instalación, el uso de RDD, DataFrame y Dataset, las APIs de programación, las librerías principales y aspectos de configuración, monitorización y optimización.
- **Módulo 3: Gestión de datos en tiempo real.** Presenta arquitecturas Lambda y Kappa, sistemas de mensajería con Kafka, Spark Streaming y Structured Streaming, así como laboratorios prácticos de integración entre Kafka y Spark.

## Componentes del repositorio

- Sitio web estático con navegación por módulos, temas y lecciones.
- Material docente en HTML para la consulta online.
- Notebooks Jupyter con ejercicios y soluciones ejecutables.
- Laboratorios prácticos con Docker, Spark, Kafka y Python.
- Autoevaluaciones asociadas a los temas.
- Recursos de estilo, scripts y configuración para despliegue web.

## Laboratorios prácticos

El proyecto incluye prácticas reproducibles que permiten experimentar con escenarios cercanos a entornos reales de procesamiento de datos masivos. Entre ellas se encuentran ejercicios de ejecución de aplicaciones Spark, uso de librerías del ecosistema Spark, producción y consumo de eventos con Kafka, procesamiento de streams con Spark Structured Streaming, validación de datos, generación de métricas, gestión de eventos erróneos y publicación de resultados enriquecidos.

## Tecnologías utilizadas

- HTML, CSS y JavaScript para la plataforma docente.
- Jupyter Notebook para prácticas guiadas.
- Apache Hadoop como infraestructura base de procesamiento distribuido.
- Apache Spark y PySpark para procesamiento batch y streaming.
- Apache Kafka para ingesta y distribución de eventos.
- Docker y Docker Compose para laboratorios reproducibles.
