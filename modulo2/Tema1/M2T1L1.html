M2T1L1 — Introducción a Apache Spark

Objetivos

  • Entender qué es Apache Spark y dónde se usa.
  • Conocer su arquitectura básica y componentes principales.
  • Ejecutar un ejemplo mínimo con PySpark.

Resumen

Apache Spark es un motor de procesamiento distribuido para big data que ofrece APIs de alto nivel y un optimizador de consultas (Catalyst). Spark soporta batch y stream processing, y provee librerías para SQL, ML, grafos y streaming.

Comandos: instalación rápida y ejecución

# Descargar y descomprimir Spark
wget https://archive.apache.org/dist/spark/spark-3.4.0/spark-3.4.0-bin-hadoop3.tgz
tar xzf spark-3.4.0-bin-hadoop3.tgz
export SPARK_HOME=$PWD/spark-3.4.0-bin-hadoop3
export PATH=$SPARK_HOME/bin:$PATH

# Ejecutar pyspark local
pyspark --master local[*]

# Ejecutar script con spark-submit
spark-submit --master local[*] my_app.py

Notas: para clúster YARN o Kubernetes ajuste --master y el deployment correspondiente.

Ejemplo PySpark (rápido)

from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('m2t1l1-demo').getOrCreate()
data = [(1,'Alice', 29),(2,'Bob', 31)]
df = spark.createDataFrame(data, ['id','name','age'])
df.show()
spark.stop()
