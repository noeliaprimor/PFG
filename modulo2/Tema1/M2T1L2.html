M2T1L2 — Componentes de la pila unificada de Spark

Objetivos

  • Conocer Spark Core, Spark SQL, MLlib, GraphX y Structured Streaming.
  • Entender cuándo usar cada librería.

Resumen

La pila de Spark provee varios módulos para cubrir diferentes necesidades: Core para ejecución, Spark SQL para datos estructurados, MLlib para ML, GraphX para grafos y Structured Streaming para streaming estructurado.

Comparativa de librerías

  • Spark Core: motor de ejecución y APIs RDD.
  • Spark SQL/DataFrames: APIs declarativas con optimización Catalyst.
  • MLlib: pipelines y algoritmos de ML.
  • GraphX: APIs para grafos y procesamiento de redes.
  • Structured Streaming: procesamiento de streams con semántica exactamente una vez.

Ejemplo: DataFrame y pipeline simple

from pyspark.sql import SparkSession
from pyspark.ml.feature import VectorAssembler
spark = SparkSession.builder.appName('m2t1l2').getOrCreate()
df = spark.createDataFrame([(0,1.0,3.0),(1,2.0,5.0)], ['id','f1','f2'])
assembler = VectorAssembler(inputCols=['f1','f2'], outputCol='features')
vec = assembler.transform(df)
vec.show()
