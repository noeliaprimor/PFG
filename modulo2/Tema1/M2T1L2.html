INTRODUCCIÓN

La pila unificada de Spark se compone de varios componentes clave que trabajan juntos para proporcionar una plataforma completa para el procesamiento de datos. Estos componentes incluyen:

  • Spark Core: Es el núcleo de Spark y proporciona las funcionalidades básicas para la gestión de tareas, la programación distribuida y el manejo de datos en memoria.
  • Spark SQL: Permite a los usuarios ejecutar consultas SQL sobre datos estructurados utilizando el motor de Spark, lo que facilita el análisis de datos tabulares.
  • Spark Streaming: Proporciona capacidades para procesar flujos de datos en tiempo real, lo que es útil para aplicaciones como la monitorización y el análisis de eventos en tiempo real.
  • Spark MLlib: Es una biblioteca de aprendizaje automático que ofrece algoritmos y herramientas para construir modelos de machine learning sobre grandes conjuntos de datos.
  • Spark GraphX: Permite el procesamiento y análisis de grafos, lo que es útil para aplicaciones como la detección de comunidades y el análisis de redes sociales.

Figura 1: Arquitectura de Spark Core como componente fundamental del motor de ejecución

Estos componentes trabajan juntos para proporcionar una plataforma unificada que puede manejar una amplia variedad de casos de uso en el ámbito del Big Data, desde el procesamiento por lotes hasta el análisis en tiempo real y el aprendizaje automático.

A continuación vamos a revisar cada uno de ellos en profundidad.

CONTENIDO

Spark Core

Spark Core es el motor de ejecución de Spark y su módulo fundamental. Proporciona las funcionalidades básicas para la gestión de tareas, la programación distribuida y el manejo de datos en memoria. Es el componente central de Spark y es responsable de la ejecución de todas las operaciones en Spark.

Se encarga de:

  • Distribuir los datos entre los nodos
  • Ejecutar tareas en paralelo
  • Gestionar memoria y almacenamiento
  • Incluye el motor de tolerancia a fallos
  • Comunicación entre los elementos
  • Planificación y ejecución de tareas

Spark Core es la base sobre la que se construyen los demás componentes de Spark, como Spark SQL, Spark Streaming, MLlib y GraphX. Proporciona las funcionalidades necesarias para ejecutar operaciones de procesamiento de datos en paralelo y manejar grandes volúmenes de datos de manera eficiente.

Spark SQL

Spark SQL es un módulo de Spark que permite a los usuarios ejecutar consultas SQL sobre datos estructurados utilizando el motor de Spark. Proporciona una interfaz de alto nivel para trabajar con datos tabulares y es compatible con una amplia variedad de fuentes de datos, como archivos CSV, JSON, Parquet, bases de datos relacionales y más.

Spark SQL se basa en el motor de ejecución de Spark Core y utiliza un optimizador llamado Catalyst para mejorar el rendimiento de las consultas. Además, Spark SQL ofrece una API de DataFrame que permite a los usuarios manipular datos estructurados de manera más intuitiva y eficiente.

Con Spark SQL, los usuarios pueden realizar operaciones como filtrado, agregación, unión y ordenamiento de datos utilizando sintaxis SQL familiar, lo que facilita el análisis de grandes conjuntos de datos estructurados.



results = spark.sql("SELECT * FROM table")
results.show()

Spark Streaming

Spark Streaming es un módulo de Spark que proporciona capacidades para procesar flujos de datos en tiempo real. Permite a los usuarios construir aplicaciones de streaming que pueden procesar datos a medida que llegan, lo que es útil para casos de uso como la monitorización, el análisis de eventos en tiempo real y la detección de fraudes.

Spark Streaming se basa en el motor de ejecución de Spark Core y utiliza un modelo de procesamiento basado en micro-batches, donde los datos se agrupan en pequeños lotes y se procesan en intervalos regulares. Esto permite a Spark Streaming manejar grandes volúmenes de datos en tiempo real con alta eficiencia.

Con Spark Streaming, los usuarios pueden conectarse a diversas fuentes de datos en tiempo real, como Kafka, Flume, sockets y más, y aplicar transformaciones y acciones sobre los flujos de datos utilizando la API de Spark.


from pyspark.sql import SparkSession
from pyspark.sql.functions import explode, split

# Crear sesión Spark
spark = SparkSession.builder \
    .appName("SimpleSparkStreaming") \
    .getOrCreate()

# Leer datos en streaming desde socket
lines = spark.readStream \
    .format("socket") \
    .option("host", "localhost") \
    .option("port", 9999) \
    .load()

# Dividir líneas en palabras
words = lines.select(
    explode(split(lines.value, " ")).alias("word")
)

# Contar palabras
wordCounts = words.groupBy("word").count()

# Mostrar resultados en consola
query = wordCounts.writeStream \
    .outputMode("complete") \
    .format("console") \
    .start()

query.awaitTermination()

Spark MLlib

Spark MLlib es una biblioteca de aprendizaje automático que ofrece algoritmos y herramientas para construir modelos de machine learning sobre grandes conjuntos de datos. Proporciona una amplia gama de algoritmos de clasificación, regresión, clustering, reducción de dimensionalidad y más, así como herramientas para la selección de características, la evaluación de modelos y la optimización de hiperparámetros.

Spark MLlib se basa en el motor de ejecución de Spark Core y está diseñado para escalar a través de clústeres de computadoras, lo que permite a los usuarios entrenar modelos de machine learning en grandes conjuntos de datos distribuidos. Además, Spark MLlib ofrece una API de alto nivel que facilita la construcción y el entrenamiento de modelos de machine learning utilizando sintaxis familiar.

Con Spark MLlib, los usuarios pueden construir modelos de machine learning para una variedad de casos de uso, como la predicción, la clasificación, el análisis de sentimientos y más, utilizando la potencia del procesamiento distribuido de Spark.


from pyspark.sql import SparkSession
from pyspark.ml import Pipeline
from pyspark.ml.feature import StringIndexer, VectorAssembler
from pyspark.ml.classification import LogisticRegression
from pyspark.ml.evaluation import BinaryClassificationEvaluator

spark = SparkSession.builder.appName("MLlibExample").getOrCreate()

# 1) Datos de ejemplo
data = [
    (25, 3000.0, "IT", 1),
    (45, 7000.0, "Sales", 0),
    (33, 5000.0, "IT", 1),
    (52, 9000.0, "HR", 0),
    (23, 2500.0, "Sales", 0),
    (41, 8000.0, "IT", 1),
]
df = spark.createDataFrame(data, ["age", "salary", "dept", "label"])

# 2) Transformaciones (features)
dept_indexer = StringIndexer(inputCol="dept", outputCol="deptIndex", handleInvalid="keep")
assembler = VectorAssembler(inputCols=["age", "salary", "deptIndex"], outputCol="features")

# 3) Modelo
lr = LogisticRegression(featuresCol="features", labelCol="label")

# 4) Pipeline
pipeline = Pipeline(stages=[dept_indexer, assembler, lr])

# 5) Train/test split y entrenamiento
train, test = df.randomSplit([0.8, 0.2], seed=42)
model = pipeline.fit(train)

# 6) Predicción
pred = model.transform(test)
pred.select("age", "salary", "dept", "label", "probability", "prediction").show(truncate=False)

# 7) Evaluación (AUC)
evaluator = BinaryClassificationEvaluator(labelCol="label", metricName="areaUnderROC")
auc = evaluator.evaluate(pred)
print("AUC =", auc)

Spark GraphX

Spark GraphX es un módulo de Spark que permite el procesamiento y análisis de grafos. Proporciona una API para trabajar con grafos distribuidos y ofrece una variedad de algoritmos para tareas como la detección de comunidades, el análisis de redes sociales, el cálculo del PageRank y más.

Spark GraphX se basa en el motor de ejecución de Spark Core y está diseñado para escalar a través de clústeres de computadoras, lo que permite a los usuarios procesar grandes grafos distribuidos. Además, Spark GraphX ofrece una API que facilita la construcción y el análisis de grafos utilizando sintaxis familiar.

Con Spark GraphX, los usuarios pueden analizar relaciones complejas entre entidades en grandes conjuntos de datos utilizando la potencia del procesamiento distribuido de Spark.



import org.apache.spark.sql.SparkSession
import org.apache.spark.graphx._
import org.apache.spark.rdd.RDD

object GraphXExample {
  def main(args: Array[String]): Unit = {

    val spark = SparkSession.builder()
      .appName("GraphXExample")
      .master("local[*]")
      .getOrCreate()

    val sc = spark.sparkContext

    // 1) Vértices (id -> nombre)
    val vertices: RDD[(VertexId, String)] = sc.parallelize(Seq(
      (1L, "Alice"),
      (2L, "Bob"),
      (3L, "Charlie"),
      (4L, "David")
    ))

    // 2) Aristas (origen -> destino, atributo = peso)
    val edges: RDD[Edge[Int]] = sc.parallelize(Seq(
      Edge(1L, 2L, 1),
      Edge(2L, 3L, 1),
      Edge(3L, 1L, 1),
      Edge(3L, 4L, 1)
    ))

    // 3) Construcción del grafo (valor por defecto para vertices que falten)
    val graph: Graph[String, Int] = Graph(vertices, edges, defaultVertexAttr = "Unknown")

    // Mostrar vértices y aristas
    println("Vertices:")
    graph.vertices.collect().foreach(println)

    println("Edges:")
    graph.edges.collect().foreach(println)

    // 4) PageRank (algoritmo clásico de ranking)
    val ranks = graph.pageRank(0.0001).vertices

    // Unimos ranks con nombres
    val ranksByName = vertices.join(ranks).map {
      case (_, (name, rank)) => (name, rank)
    }

    println("PageRank:")
    ranksByName.collect().foreach(println)

    spark.stop()
  }
}
