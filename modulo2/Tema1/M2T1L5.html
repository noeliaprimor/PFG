Ejercicio propuesto

M2T1L5 — Instalación y configuración de Spark

Objetivos

  • Instalar Spark en modo local y comprender variables de entorno.
  • Configurar el entorno para ejecutar PySpark y spark-submit.

Pasos de instalación (resumen)

  1. Instalar Java (OpenJDK 11+): sudo apt install openjdk-11-jdk
  2. Descargar Spark precompilado y descomprimir.
  3. Configurar SPARK_HOME y añadir $SPARK_HOME/bin al PATH.
  4. Probar con pyspark --master local[*].

Configuraciones importantes

  • spark.executor.memory — memoria por executor.
  • spark.driver.memory — memoria del driver.
  • Particionado: rdd.repartition(n) o df.repartition(n).
