M2T1L7 — Ejecución de ejemplos de Spark

Objetivos

  • Aprender a ejecutar aplicaciones con spark-submit.
  • Conocer parámetros comunes (master, deploy-mode, conf).

Ejecutar ejemplo

# Ejecutar script Python
spark-submit --master local[*] ejemplo.py

# Ejecutar ejemplo en cluster (YARN)
spark-submit --master yarn --deploy-mode cluster ejemplo.py
