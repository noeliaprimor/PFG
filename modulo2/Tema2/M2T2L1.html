DESCRIPCIÓN

En esta lección revisamos los entornos de desarrollo y ejecución para aplicaciones Spark: shells interactivas, proyectos empaquetados (sbt/maven para Scala, pip/virtualenv para Python), contenedores y despliegue en clústeres (YARN, Kubernetes, Standalone).

CONTENIDOS

Entornos locales y remotos

Desarrollo local con pyspark o spark-shell para prototipado. Empaquetado con spark-submit para producción. Uso de contenedores Docker para reproducibilidad.

Estructura de proyecto (Python)

myproject/
  app/
    __init__.py
    job.py
  requirements.txt
  setup.py
  Dockerfile

Estructura de proyecto (Scala - sbt)

project/
build.sbt
src/main/scala/com/example/MyApp.scala

COMANDOS Y EJEMPLOS

Instalar PySpark en virtualenv

python3 -m venv venv
source venv/bin/activate
pip install pyspark

Ejecutar con spark-submit

/opt/spark/bin/spark-submit --master yarn --deploy-mode cluster --conf spark.executor.memory=2g myapp.py
