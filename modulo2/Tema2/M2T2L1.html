Ejercicio propuesto

Objetivos

  • Conocer cómo organizar un proyecto Spark (módulos, dependencias).
  • Configurar entornos locales y de producción.

Resumen

Al desarrollar aplicaciones Spark es importante estructurar el proyecto, gestionar dependencias (sbt / pip) y preparar el despliegue (spark-submit, contenedores, clusters).

Comandos y ejemplos

# Crear virtualenv para dependencias Python
python3 -m venv venv
source venv/bin/activate
pip install pyspark

# Ejecutar app
python my_app.py
