M2T2L2 — Desarrollo y ejecución de una aplicación Spark

Objetivos

  • Conocer la estructura mínima de una aplicación Spark en Python y Scala.
  • Aprender a probar localmente y desplegar con spark-submit.

Estructura de proyecto (Python)

my_app/
  ├─ src/
  │   └─ app.py
  ├─ requirements.txt
  └─ Dockerfile

# app.py
from pyspark.sql import SparkSession
if __name__ == '__main__':
    spark = SparkSession.builder.appName('my_app').getOrCreate()
    # lógica
    spark.stop()

Ejemplo rápido

from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('m2t2l2').getOrCreate()
df = spark.createDataFrame([(1,'x'),(2,'y')], ['id','val'])
df.show()
spark.stop()

Notebook: M2T2L2.ipynb

Despliegue

Usar spark-submit o crear una imagen Docker con Spark y lanzar en Kubernetes.
