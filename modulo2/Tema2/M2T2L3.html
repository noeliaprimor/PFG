Creación de sesiones: SparkSession y SparkContext

  • Descripción
  • API y ejemplos
  • Configuración
  • Ejercicios

DESCRIPCIÓN

Entender la diferencia entre SparkContext (nivel bajo) y SparkSession (API unificada para SQL/DataFrame) y cómo inicializarlas correctamente según entorno (local, cluster).

API y ejemplos

PySpark — SparkSession

from pyspark.sql import SparkSession
spark = SparkSession.builder \
    .appName("MiApp") \
    .config("spark.some.config", "value") \
    .getOrCreate()

# Usar spark para leer un CSV
df = spark.read.csv('/data/archivo.csv', header=True, inferSchema=True)
df.show()

Scala — SparkContext y SparkSession

import org.apache.spark.sql.SparkSession
val spark = SparkSession.builder.appName("MiApp").getOrCreate()
val sc = spark.sparkContext

CONFIGURACIÓN

Parámetros clave: spark.executor.memory, spark.driver.memory, spark.executor.cores, y paralelismo por defecto spark.default.parallelism.

EJERCICIOS

  1. Inicializa una SparkSession con distintos parámetros y observa el impacto en tiempos de ejecución con un pequeño dataset.
  2. Explica por qué SparkSession reemplaza a SparkContext en las APIs modernas.
