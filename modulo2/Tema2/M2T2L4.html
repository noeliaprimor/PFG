Inicialización de Spark en Scala y Python

  • Descripción
  • Scala
  • Python
  • Despliegue

DESCRIPCIÓN

Aprenderemos a crear sesiones y contextos en Scala y Python, configurar parámetros, y preparar el entorno para pruebas locales y despliegue en clúster.

SCALA: Ejemplo mínimo

import org.apache.spark.sql.SparkSession
object App {
  def main(args: Array[String]): Unit = {
    val spark = SparkSession.builder().appName("ScalaApp").master("local[*]").getOrCreate()
    val df = spark.read.json("data/example.json")
    df.show()
    spark.stop()
  }
}

PYTHON: Ejemplo mínimo

from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('PyApp').master('local[*]').getOrCreate()
df = spark.read.json('data/example.json')
df.show()
spark.stop()

DESPLIEGUE

Para producción empaqueta jar (Scala) o usa --py-files/--archives para Python. Ajusta recursos con opciones de spark-submit y usa el UI para depuración.
