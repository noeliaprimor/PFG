Cargar y guardar datos en Spark

  • Descripción
  • Lectura

DESCRIPCIÓN

Aprenderemos las APIs para leer/escribir CSV, JSON, Parquet y conectar con HDFS, S3 o bases de datos JDBC, así como buenas prácticas de particionado y esquemas.

LECTURA

# CSV
df = spark.read.option('header', True).csv('/data/my.csv')
# Parquet
pq = spark.read.parquet('/data/parquet/')
# JDBC
jdbc = spark.read.format('jdbc').option('url','jdbc:postgresql://host/db').option('dbtable','table').option('user','u').option('password','p').load()

ESCRITURA

# escribir Parquet particionado
df.write.mode('overwrite').partitionBy('year','month').parquet('/out/parquet')
# escribir CSV
df.write.mode('overwrite').option('header',True).csv('/out/csv')

FORMATOS Y PARTICIONADO

Parquet es preferido por su compresión y esquema. El particionado mejora consultas filtradas pero puede generar demasiados ficheros si los buckets son pequeños.
