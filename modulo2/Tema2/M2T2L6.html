Colecciones de datos: RDD, DataFrame y Dataset

  • Descripción
  • Comparativa
  • Ejemplos

DESCRIPCIÓN

RPM: RDD es la API de bajo nivel; DataFrame y Dataset proporcionan abstracciones con esquema y optimizaciones (Catalyst).

COMPARATIVA

  • RDD: control completo pero sin optimizaciones automáticas.
  • DataFrame: esquema, optimizaciones y API declarativa.
  • Dataset: tipado (Scala/Java) y optimizaciones.

EJEMPLOS

# DataFrame
df = spark.read.json('/data/events.json')
df.filter(df.value > 100).select('id','value')

# RDD
rdd = sc.textFile('/data/file.txt').map(lambda r: r.split(','))
