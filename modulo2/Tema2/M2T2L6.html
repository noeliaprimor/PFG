M2T2L6 — Colecciones de datos: RDD, DataFrame y Dataset

Objetivos

  • Entender diferencias y cuándo escoger RDD vs DataFrame vs Dataset.

Ejemplo comparativo

# RDD
from pyspark import SparkContext
sc = SparkContext()
rdd = sc.parallelize([('a',1),('b',2)])
print(rdd.map(lambda x: x[1]).collect())
sc.stop()

# DataFrame
from pyspark.sql import SparkSession
spark = SparkSession.builder.getOrCreate()
df = spark.createDataFrame([('a',1),('b',2)], ['k','v'])
df.select('v').show()
