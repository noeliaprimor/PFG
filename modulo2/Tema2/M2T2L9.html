Operaciones y optimizaciones sobre DataFrames y Datasets

  • Optimizador Catalyst
  • Persistencia
  • Broadcast joins

OPTIMIZADOR CATALYST

Catalyst aplica reglas de optimización al DAG, reorganiza joins, proyecta columnas y elimina operaciones redundantes. Usar DataFrame API permite aprovechar estas optimizaciones.

PERSISTENCIA Y CACHE

Usa df.persist() o df.cache() cuando reutilices un conjunto de datos para evitar recomputaciones costosas.

BROADCAST JOINS

Para joins entre una gran y una pequeña tabla, usa broadcast para evitar shuffle: broadcast(smallDF).
