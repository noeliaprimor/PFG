DESCRIPCIÓN

Apache Spark ofrece una pila de librerías que extienden su motor central (Spark Core) para cubrir casos de uso concretos: Spark SQL para datos estructurados, MLlib para machine learning, GraphX para grafos y Structured Streaming para flujos. Además existen paquetes externos y conectores (JDBC, Kafka, Cassandra, Parquet, etc.)

COMPONENTES PRINCIPALES

Spark SQL / DataFrame API

Permite trabajar con datos tabulares, usar SQL y beneficiarse del optimizador Catalyst y Tungsten para alto rendimiento.

Spark MLlib

Biblioteca de aprendizaje automático con algoritmos escalables (classification, regression, clustering, collaborative filtering).

Spark GraphX

Procesamiento y análisis de grafos (PageRank, connected components, etc.).

Spark Structured Streaming

Procesamiento de streams con abstracción de DataFrame/SQL y garantías de consistencia.

Conectores y paquetes

Conectores para Kafka, JDBC, Cassandra, S3, Elasticsearch y paquetes del ecosistema en spark-packages.org.

EJEMPLOS RÁPIDOS

# Spark SQL
from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('libs').getOrCreate()
df = spark.read.json('/data/examples.json')
df.createOrReplaceTempView('v')
res = spark.sql('SELECT col, count(*) FROM v GROUP BY col')
res.show()
