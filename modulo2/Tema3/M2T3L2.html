Spark SQL: gestión de datos estructurados

  • DataFrames

DESCRIPCIÓN

Spark SQL proporciona la abstracción de DataFrame y un motor SQL optimizado (Catalyst). Permite ejecutar consultas SQL y combinar el poder de las APIs declarativas con las optimizaciones internas.

DATAFRAMES

Los DataFrames representan tablas con esquema. Son inmutables y permiten aplicar transformaciones declarativas que Catalyst optimizará.

df = spark.read.option('header',True).csv('/data/my.csv')
df.printSchema()
df.select('col1','col2').where(df.col3 > 100).show()

USO DE SQL

Puedes registrar vistas temporales y ejecutar consultas SQL:

df.createOrReplaceTempView('v')
res = spark.sql('SELECT col, COUNT(*) FROM v GROUP BY col')
res.show()

OPTIMIZACIÓN

Consejos: especifica esquemas, usa Parquet, evita UDFs costosos, y usa particionado apropiado. Revisa el plan con explain().

df.select('col').explain(True)
