GraphX: procesamiento y análisis de grafos

  • Descripción
  • API básica
  • Algoritmos

DESCRIPCIÓN

GraphX es la librería de Spark para grafos; combina RDDs y una API de alto nivel para representar vértices y aristas y ejecutar algoritmos distribuidos (PageRank, connected components, etc.).

API BÁSICA (Scala)

import org.apache.spark.graphx._
val vertices = sc.parallelize(Seq((1L, "Alice"), (2L, "Bob")))
val edges = sc.parallelize(Seq(Edge(1L,2L,1)))
val g = Graph(vertices, edges)

ALGORITMOS COMUNES

  • PageRank
  • Connected Components
  • Triangle Count

Estos algoritmos se ejecutan de forma distribuida y se usan para recomendaciones, detección de comunidades y análisis de relaciones.
