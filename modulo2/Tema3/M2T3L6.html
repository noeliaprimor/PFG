DESCRIPCIÓN

Esta lección recopila recursos para profundizar en las librerías principales de Spark: Spark SQL, GraphX, Structured Streaming y MLlib. El objetivo es entender cuándo usar cada componente y cómo se integra con el resto de la plataforma.

La selección combina documentación oficial, guías de programación, referencias de APIs y lecturas útiles para resolver casos prácticos con datos estructurados, grafos, flujos y modelos de aprendizaje automático.

CONTENIDO

Vista general de librerías Spark

  • Apache Spark Documentation: índice oficial de SQL, streaming, MLlib, GraphX, APIs y despliegue.
  • PySpark API Reference: referencia de clases y funciones para Python.
  • Scala API Docs: documentación de APIs Scala, especialmente útil para Dataset y GraphX.

Spark SQL

  • Spark SQL, DataFrames and Datasets Guide: consultas SQL, DataFrames, Datasets, vistas temporales y catálogo.
  • Spark SQL Reference: sintaxis SQL, funciones, tipos de datos y comportamiento del motor SQL.
  • Spark SQL Performance Tuning: optimización Catalyst, AQE, particionado, caché y joins.

GraphX y análisis de grafos

  • GraphX Programming Guide: guía de grafos, vértices, aristas, triplets, PageRank y componentes conectados.
  • GraphX Scala API: referencia de clases, operadores y algoritmos de GraphX.
  • Pregel API: patrón de cómputo iterativo sobre grafos para algoritmos distribuidos.

Structured Streaming

  • Structured Streaming Programming Guide: modelo de stream como tabla, microbatches, checkpoints y tolerancia a fallos.
  • Streaming APIs on DataFrames and Datasets: operaciones soportadas, ventanas, joins, triggers y output modes.
  • Structured Streaming + Kafka Integration: lectura y escritura de eventos Kafka desde Spark.

MLlib

  • MLlib Main Guide: pipelines de machine learning, transformers, estimators y evaluadores.
  • Classification and Regression: algoritmos supervisados, parámetros y ejemplos.
  • Clustering: algoritmos no supervisados como K-means y Gaussian Mixture.
  • ML Tuning: validación cruzada, búsqueda de hiperparámetros y evaluación.

Lecturas complementarias

  • Learning Spark, 2nd Edition: recorrido práctico por SQL, streaming y machine learning.
  • Graph Algorithms: lectura útil para entender PageRank, comunidades y caminos en grafos.
  • Spark Packages: paquetes externos para conectar Spark con otros sistemas.

RUTA RECOMENDADA

  1. Empezar por Spark SQL, porque es la base de DataFrames y Structured Streaming.
  2. Revisar GraphX si el problema puede modelarse como relaciones entre entidades.
  3. Estudiar Structured Streaming antes de integrar Kafka o procesar eventos.
  4. Practicar MLlib con pipelines reproducibles y evaluación de modelos.
  5. Comparar qué componente resuelve mejor cada caso de uso del ejercicio final.
