OBJETIVOS DE APRENDIZAJE:

  1. Reconocer las librerías principales de Spark y su papel en soluciones de datos distribuidos.
  2. Aplicar Spark SQL para consultas, transformaciones y análisis de datos estructurados.
  3. Comprender los fundamentos del procesamiento de grafos con GraphX.
  4. Introducir pipelines de streaming con Spark Streaming y Structured Streaming.
  5. Conocer el flujo básico de modelado con MLlib para tareas de machine learning.

PRE-REQUISITOS:

Se recomienda haber completado el Tema 2 y dominar operaciones básicas con DataFrames y RDD. Es conveniente tener nociones de SQL y fundamentos de estadística descriptiva para aprovechar mejor los apartados de Spark SQL y MLlib.
