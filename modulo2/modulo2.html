Módulo 2: Procesamiento Paralelo en memoria: Apache Spark


Introducción al módulo

En este módulo se introduce Apache Spark, una plataforma de procesamiento de datos en memoria que permite realizar análisis rápidos y eficientes sobre grandes volúmenes de datos. Se cubren los conceptos fundamentales, la arquitectura, las APIs principales (RDDs, DataFrames, Datasets) y las mejores prácticas para desarrollar aplicaciones Spark.
