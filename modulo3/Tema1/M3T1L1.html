DESCRIPCIÓN

Este tema presenta las bases del procesamiento en tiempo real: ingestión continua de eventos, procesamiento con baja latencia y entrega de resultados.

CONTENIDO

Capítulo X: Introducción al procesamiento en tiempo real en Apache Spark

Introducción

El procesamiento en tiempo real en Apache Spark permite analizar y transformar datos a medida que se generan.

Conceptos clave

Evento

Unidad de información generada por un sistema externo.

Ejemplo básico en Structured Streaming (PySpark)

from pyspark.sql import SparkSession

spark = SparkSession.builder.appName("StreamingExample").getOrCreate()

stream_df = (spark
    .readStream
    .format("socket")
    .option("host", "localhost")
    .option("port", 9999)
    .load())

result = stream_df.groupBy("value").count()

query = (result
    .writeStream
    .outputMode("complete")
    .format("console")
    .start())

query.awaitTermination()

Ventanas temporales

from pyspark.sql.functions import window

windowed = (stream_df
    .groupBy(window(stream_df.timestamp, "10 minutes"))
    .count())

Watermarking

stream_df.withWatermark("timestamp", "5 minutes")
CONCLUSIÓN

Structured Streaming ofrece un modelo declarativo que facilita el desarrollo y la integración con Spark SQL.
