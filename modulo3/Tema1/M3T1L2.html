M3T1L2 — Arquitecturas Lambda y Kappa

Objetivos

  • Entender la diferencia entre Lambda y Kappa y sus ventajas.
  • Diseñar cuándo usar cada una según requisitos de consistencia y latencia.

Resumen

La arquitectura Lambda mezcla batch y speed layer: una capa de procesamiento por lotes para procesado histórico y una capa de velocidad para datos recientes. Kappa propone un único pipeline de streaming que reprocese el historial si es necesario usando la misma lógica.
