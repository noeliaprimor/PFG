M3T1L3 — Introducción a Kafka

Objetivos

  • Conocer la arquitectura básica de Kafka: brokers, topics, particiones y zookeper (o KRaft).
  • Entender productores, consumidores y conceptos de offset y retención.

Resumen

Kafka es una plataforma de mensajería distribuida diseñada para transmitir grandes volúmenes de datos con alta disponibilidad. Los topics se particionan y replican entre brokers; los consumidores gestionan offsets para garantizar la lectura correcta de eventos.

Comandos básicos (con Kafka instalado)

# crear topic
$KAFKA_HOME/bin/kafka-topics.sh --create --topic prueba --bootstrap-server localhost:9092 --partitions 3 --replication-factor 1

# lanzar consumidor de consola
$KAFKA_HOME/bin/kafka-console-consumer.sh --topic prueba --bootstrap-server localhost:9092 --from-beginning
