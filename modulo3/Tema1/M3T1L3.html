DESCRIPCIÓN

Apache Kafka es una plataforma de mensajería distribuida para el manejo de streams de eventos con alta durabilidad y throughput. Se usa como sistema de ingestión y bus de eventos en arquitecturas de streaming.

CONCEPTOS CLAVE

  • Topic: canal lógico para mensajes.
  • Partition: permite paralelismo y orden por partición.
  • Producer/Consumer: productores envían, consumidores leen offsets.
  • Retention: periodo de almacenamiento.

COMANDOS ÚTILES

# listar topics
$KAFKA_HOME/bin/kafka-topics.sh --list --bootstrap-server localhost:9092

# crear topic
$KAFKA_HOME/bin/kafka-topics.sh --create --topic ejemplo --partitions 3 --replication-factor 1 --bootstrap-server localhost:9092

EJERCICIO
