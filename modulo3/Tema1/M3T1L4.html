DESCRIPCIÓN

Presentaremos las diferencias entre DStreams (antiguo) y Structured Streaming (moderno), y las capacidades para ventanas, watermark y semánticas de entrega (al menos una vez / exactamente una vez).

MODELOS DE PROCESAMIENTO

Structured Streaming trata el stream como una tabla en crecimiento; soporta operaciones declarativas y se integra con DataFrame API.

EJEMPLO

from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('m3t1l4').getOrCreate()
df = spark.readStream.format('kafka').option('kafka.bootstrap.servers','localhost:9092').option('subscribe','events').load()
# parse, transform, write stream
