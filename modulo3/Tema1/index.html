Objetivos de Aprendizaje (PTE):

  1. Entender los patrones arquitectónicos para procesamiento en tiempo real (Lambda vs Kappa).
  2. Conocer las herramientas clave (Kafka y Spark Structured Streaming) y cuándo aplicarlas.
  3. Ser capaz de diseñar un pipeline básico de ingestión y procesamiento de eventos.
  4. Identificar requisitos de latencia, consistencia y tolerancia a fallos para sistemas de streaming.

PRE-REQUISITOS:(PTE)

Conocimientos básicos de sistemas distribuidos y fundamentos de Spark/Python. Es recomendable disponer de un entorno con Docker o acceso a un cluster mínimo para prácticas.
