OBJETIVOS DE APRENDIZAJE:

  1. Diferenciar los enfoques Lambda y Kappa para arquitecturas de datos en tiempo real.
  2. Comprender los fundamentos de Kafka y Spark Streaming dentro de un pipeline de eventos.
  3. Diseñar un flujo básico de ingesta y procesamiento con requisitos de baja latencia.
  4. Identificar implicaciones de consistencia, tolerancia a fallos y escalabilidad en sistemas de streaming.

PRE-REQUISITOS:

Se recomienda haber completado el Módulo 2 (especialmente APIs de Spark), junto con conocimientos básicos de sistemas distribuidos y programación en Python o Scala. Es conveniente contar con Docker o un entorno de laboratorio para realizar prácticas.
