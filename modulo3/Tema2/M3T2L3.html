KAFKA CONNECT

Framework para integrar Kafka con sistemas externos mediante connectors: source (extraen datos) y sink (escriben datos).

SINKS COMUNES

  • S3 / HDFS (almacenamiento en ficheros)
  • JDBC (bases relacionales)
  • Elasticsearch (búsqueda y analítica)

EJEMPLO: CONFIGURACIÓN BÁSICA

{
  "name": "s3-sink",
  "config": {
    "connector.class": "io.confluent.connect.s3.S3SinkConnector",
    "tasks.max": "1",
    "topics": "events",
    "s3.bucket.name": "my-bucket"
  }
}
