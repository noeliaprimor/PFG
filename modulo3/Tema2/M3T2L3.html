M3T2L3 — Sinks y conectores

Objetivos

  • Conocer conectores comunes: Kafka Connect, Flink connectors, y sinks como S3, HDFS, Elasticsearch.

Resumen

Para integrar sistemas en un pipeline de eventos, Kafka Connect facilita la extracción y carga a destinos (sinks). Existen conectores para S3, HDFS, Elasticsearch, y bases de datos; seleccionado según latencia y formato requerido.

Ejemplo: configuración básica de un sink (conceptual)

{
  "name": "s3-sink",
  "config": {
    "connector.class": "io.confluent.connect.s3.S3SinkConnector",
    "tasks.max": "1",
    "topics": "topic",
    "s3.bucket.name": "my-bucket"
  }
}
