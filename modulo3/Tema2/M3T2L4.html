DESCRIPCIÓN

Esta lección recopila recursos para profundizar en los contenidos del tema: adquisición de eventos, Apache Kafka, integración con Spark Structured Streaming, conectores, sinks, Kafka Connect, CDC, formatos de serialización y buenas prácticas de operación.

La selección combina documentación oficial, guías prácticas y referencias útiles para reforzar el ejercicio propuesto del tema, especialmente el flujo productor-consumidor, la escritura en sinks y el uso de checkpoints.

CONTENIDO

Documentación oficial imprescindible

  • Apache Kafka Documentation: referencia principal para brokers, topics, productores, consumidores, replicación, configuración, seguridad y operación del clúster.
  • Spark Structured Streaming Programming Guide: guía oficial del modelo de programación, microbatches, ventanas, watermarks, output modes, triggers y tolerancia a fallos.
  • Structured Streaming + Kafka Integration Guide: referencia específica para leer desde Kafka y escribir resultados en topics Kafka usando Spark.
  • Kafka Connect: explicación de Kafka Connect como herramienta para mover datos de forma escalable entre Kafka y sistemas externos.

Kafka en detalle

  • Apache Kafka Introduction: visión general de Kafka, su modelo distribuido y los casos de uso habituales.
  • Producer Configs: recurso para revisar parámetros como acks, idempotencia, batching, compresión y reintentos.
  • Consumer Configs: referencia para entender opciones de consumidores, commits de offsets, grupos y comportamiento de lectura.
  • Topic Configs: documentación de retención, compactación, segmentos y configuraciones clave de topics.

Spark Structured Streaming

  • APIs on DataFrames and Datasets: operaciones disponibles sobre flujos, agregaciones, ventanas y consultas continuas.
  • Watermarking: apartado específico para eventos tardíos, tiempo de evento y limpieza de estado.
  • Output Modes: explicación de los modos append, update y complete.
  • Triggers: referencia para controlar la frecuencia de ejecución de las consultas streaming.

Conectores, sinks y Kafka Connect

  • Kafka Connectors for Confluent Platform: catálogo de conectores habituales, incluyendo JDBC, Elasticsearch, S3 y HDFS.
  • JDBC Source and Sink Connector: documentación para mover datos entre Kafka y bases de datos relacionales mediante JDBC.
  • Amazon S3 Sink Connector: referencia para exportar topics Kafka a almacenamiento S3 usando formatos como Avro, JSON o Parquet según configuración.
  • Spark Parquet Data Source: guía para lectura y escritura en Parquet, formato recomendado para analítica.

CDC y eventos desde bases de datos

  • Debezium Documentation: documentación de referencia para capturar cambios de bases de datos como eventos.
  • Debezium Architecture: explicación de cómo Debezium se despliega normalmente sobre Kafka Connect para crear pipelines CDC.
  • Debezium Tutorial: tutorial práctico para entender un flujo CDC completo con Kafka.

Formatos, esquemas y contratos de datos

  • Apache Avro Documentation: formato binario orientado a esquemas, frecuente en pipelines Kafka.
  • Protocol Buffers Overview: referencia de Protobuf como alternativa eficiente para contratos estrictos de datos.
  • JSON Schema Getting Started: recurso para validar eventos JSON y documentar su estructura.
  • Schema Registry: documentación para gestionar compatibilidad y evolución de esquemas.

Práctica y entornos locales

  • Apache Kafka GitHub: código fuente, ejemplos y referencias del proyecto Kafka.
  • Apache Spark GitHub: repositorio oficial de Spark, útil para ejemplos, issues y documentación técnica.
  • Spark Packages: buscador de paquetes y conectores para ampliar aplicaciones Spark.
  • Docker Compose Documentation: recurso práctico para levantar entornos locales con Kafka, Kafka Connect, bases de datos y servicios auxiliares.
