M3T3L1 — Spark Streaming

Objetivos

  • Comprender el modelo de Structured Streaming y cómo se integra con sistemas de mensajería.

Resumen

Spark Structured Streaming permite construir pipelines de procesamiento continuo usando la API de DataFrame/SQL; soporta fuentes como Kafka y sinks variados. Soporta semánticas de procesamiento al menos una vez o exactamente una vez según el sink y la configuración.

Ejemplo

from pyspark.sql import SparkSession
spark = SparkSession.builder.appName('m3t3l1').getOrCreate()
df = spark.readStream.format('kafka').option('kafka.bootstrap.servers','localhost:9092').option('subscribe','topic').load()
