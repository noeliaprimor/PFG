Integración con Kafka

LEER DESDE KAFKA

Usar spark.readStream.format('kafka') y configurar bootstrap.servers y topics. El campo value viene en binario y debe convertirse a string o bytes y parsearse.

PARSEAR JSON

from pyspark.sql.functions import from_json, col
schema = 'id INT, value STRING'
df2 = df.selectExpr("CAST(value AS STRING) as json_str")
df_parsed = df2.select(from_json(col('json_str'), schema).alias('data')).select('data.*')
