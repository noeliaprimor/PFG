Módulo 3: Gestión de datos en tiempo real


Introducción al módulo

En este módulo se estudian arquitecturas y herramientas para el procesamiento de datos en tiempo real. Se cubren patrones arquitectónicos (Lambda y Kappa), sistemas de mensajería como Apache Kafka, y el uso de Spark Structured Streaming para procesar y enriquecer flujos de datos con garantías de tolerancia y escalabilidad.
